## Overview

Social web app that allows users to create, share, and browse top five lists for movies and TV.

## Technologies Used

- **Next.js**: For fast, server-rendered React applications.
- **NextUI**: Pretty slick react UI library
- **TailwindCSS**: Rapid styling
- **Prisma**: Great ORM
- **Clerk**: Cloud auth
- **React Query**: The gold standard of client-side data fetching
